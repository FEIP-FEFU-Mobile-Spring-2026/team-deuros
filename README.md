
# Deuros 
## Описание
---
Deuros — это мобильное приложение интернет-магазина одежды для платформы Android. Позволяет просматривать каталог товаров, фильтровать по категориям и оформлять заказы.

## Стек технологий
---
| Категория | Технология |
|-----------|------------|
| Платформа | Android (minSdk 24) |
| Язык | Kotlin |
| UI | Jetpack Compose + Material 3 |
| Сборка | Gradle (Kotlin DSL) |

## Сборка и запуск
---
### Требования
- Android Studio Hedgehog (2023.1.1) или новее
- JDK 17 (рекомендуется для Android Studio)
- Android SDK Platform 36 с API Level 36 extension 1
- Эмулятор или устройство с Android 7.0+ (API 24)
